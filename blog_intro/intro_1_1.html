[Information Flush] Low Rank Feature in Model Matrices & Coastal Plains Estuaries


What is SVD

SVD stands for Singular Value Decomposition (not Stable Video Diffusion). When we perform SVD on a matrix W, we obtain three matrices: U, S, and V.

  • Singular value indicates the degree of stretching or compression of the matrix transformation, affecting the size of the input vector projection in the output space.
  • Left singular vector Represents the important direction in the input space (or original space).
  • Right singular vector Represents the important direction in the output space (or transformed space.



    • Low Rank Feature in Model Matrices

    LLM weight matrices present long-tailed shape, which means that they are low rank. VLM, VAR, VLA also have this kind of feature.



    Information Flush: Pre-training ↔ Delta

    After pre-training, the singular values of the weight matrix form a long-tailed shape, like the delta of a Coastal Plain Estuary[1]. To put it more vividly, it's the information flush[2] that changed the weights. And what kind of information is this?

    It's like when we first learn a language — we focus on high-frequency words and expressions because they help us use the language effectively.

    Similarly, after pre-training, the singular values show us which features are important for the model to finish tasks. Generally, the larger the singular value, the greater its impact on task performance.
